# node-server-calc
> Calculator as a service ➗✖️➕➖

---
## Install
1. `git clone https://github.com/madeagency/node-server-calc.git`
2. `npm install`

---
## Start (development)
`npm start`

---
## Start (production)
`npm run build; npm run serve`

---
## Test
`npm test`

---
## Requirementss
- Written in `Babel`
- Uses `express` and `body-parser`
